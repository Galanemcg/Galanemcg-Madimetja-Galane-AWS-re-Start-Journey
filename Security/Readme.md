**Readme**


This labs provides hands-on experience with various AWS security and monitoring services, including AWS KMS, AWS Encryption CLI, Amazon Inspector, Amazon SNS, and Amazon CloudWatch.

In this labs i have learned how to:
1. Create an AWS KMS encryption key: I have created a new AWS KMS encryption key to demonstrate encryption and decryption capabilities.
2. Install the AWS Encryption CLI: I have installed the AWS Encryption CLI to encrypt and decrypt plaintext data.
3. Encrypt plaintext: I have used the AWS Encryption CLI to encrypt plaintext data using the KMS key created earlier.
4. Decrypt ciphertext: I have used the AWS Encryption CLI to decrypt the ciphertext generated.
5. Patch Linux instances using default baseline: I have demonstrate patching Linux instances using the default patch baseline.
6. Create custom patch baseline: I have create a custom patch baseline to demonstrate advanced patching capabilities.
7. Use patch groups to patch Windows instances using custom patch baseline: I have use patch groups to patch Windows instances using the custom patch baseline.
8. Verify patch compliance: I have verify the patch compliance of the instances patched in the previous steps.
9. Activate Amazon Inspector: I have activated Amazon Inspector to analyze and interpret vulnerability findings.
10. Analyze and interpret vulnerability findings: I have analyzed and interpret the vulnerability findings generated by Amazon Inspector.
11. Remediate the vulnerabilities found by Amazon Inspector: I have remediate the vulnerabilities found by Amazon Inspector.
12. Create an Amazon SNS notification: I have create an Amazon SNS notification to demonstrate alerting capabilities.
13. Configure a CloudWatch alarm: I have configure a CloudWatch alarm to trigger the SNS notification created.
14. Stress test an EC2 instance: I have stress test an EC2 instance to trigger the CloudWatch alarm.
15. Confirm that an Amazon SNS email was sent: I have confirm that an Amazon SNS email was sent as a result of the CloudWatch alarm.
16. Create a CloudWatch dashboard: I have created a CloudWatch dashboard to visualize metrics and alarms.
